## Installation



Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/szuviria/eCommerce-reactjs.git
$ cd ecommercereact
$ npm install
$ npm run dev
```

For production environments...
```sh
See .env.example 
VITE_apiKey=
VITE_authDomain=
VITE_projectId=
VITE_storageBucket=
VITE_messagingSenderId=
VITE_appId=
```
